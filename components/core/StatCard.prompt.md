**StatCard** — a single record figure (huge Anton numeral + uppercase label). Lay several in a hairline-divided row for the fight record.

```jsx
<StatCard value="30" label="Combats" />
<StatCard value="20" label="Victoires" tone="accent" />
<StatCard value="10" label="Défaites" tone="faint" />
```

`tone`: `default | accent | faint`.
