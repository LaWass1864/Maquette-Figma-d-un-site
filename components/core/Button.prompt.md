**Button** — the system's call to action; square corners, uppercase, lifts with an orange shadow on hover (`ghost` instead shifts its outline to accent).

```jsx
<Button href="#sponsoring" size="lg" icon="→">Devenir partenaire</Button>
<Button variant="ghost">En savoir plus</Button>
<Button variant="nav">Devenir partenaire</Button>
```

Variants: `primary` (default, accent fill) · `ghost` (hairline outline) · `nav` (compact header). Sizes `sm | md | lg`. Pass `href` to render an anchor.
